# README.md

# 🤖 SovereignSynth
### *The Autonomous Digital Twin & Creator Economy Manager*

[![Solana](https://img.shields.io/badge/Made%20for-Solana-14F195?style=for-the-badge&logo=solana)](https://solana.com/)
[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)
[![AI-Powered](https://img.shields.io/badge/AI-Agentic-8B5CF6?style=for-the-badge&logo=openai)](https://openai.com/)

**SovereignSynth** empowers creators and brands by deploying autonomous "Digital Twins"—AI agents capable of negotiating deals, managing on-chain treasuries, and distributing rewards to fans automatically on the Solana blockchain.

## ✨ Key Features

-   **🧠 Autonomous Digital Twin:** A persistent AI agent that represents your brand 24/7, learning from your preferences and market trends.
-   **🤝 AI-Driven Negotiations:** Automated deal-flow processing with brands (e.g., HyperX, RedBull), handling initial proposals and value assessments.
-   **💎 Transparent Treasury:** Integrated Solana wallet support to track volume, earnings, and payouts in real-time.
-   **🎁 Automated Fan Rewards:** High-fidelity engagement tracking that triggers SOL distributions to top community members without manual intervention.
-   **⚡ High-Performance UI:** A sleek, "Synth-wave" inspired dashboard built with Vite, Tailwind CSS, and Framer Motion for smooth interactions.

## 🛠️ Tech Stack

-   **Frontend:** React, TypeScript, Tailwind CSS, Framer Motion, Lucide-React.
-   **Backend:** Node.js (Express), TSX.
-   **Blockchain:** @solana/web3.js, Wallet Adapter React.
-   **AI Intelligence:** OpenAI API (Integration-ready).

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- A Solana Wallet (Phantom, Solflare, etc.)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/sovereign-synth.git
   cd sovereign-synth
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env` file in the root (refer to `.env.example`):
   ```env
   OPENAI_API_KEY=your_key_here
   SOLANA_RPC_URL=https://api.mainnet-beta.solana.com
   ```

4. **Run the Development Server:**
   ```bash
   # Terminal 1: Start the Backend
   npm run server

   # Terminal 2: Start the Frontend
   npm run dev
   ```
